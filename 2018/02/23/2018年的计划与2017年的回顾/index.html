2018年的计划与2017年的回顾

发表于 2018-02-23 | 分类于 虎嗅蔷薇 |

2017是颓废的一年，所以我只能把希望寄托在2018年。。。。

关于2017年

下面这张图也许可以说明点什么~~~

过去的一年真的可以说懈怠了好多，主要体现在博客停更，几乎没怎么学习新技术，年初的计划到了年末还是没能完成。当然这里面有客观原因，最主要的还是个人问题。年初换了新工作，本来想继续在Android开发的道路上开疆扩土，结果由于项目上的原因（主要是人满了）进了公司做了3个月的测试，说实话，那段时间简直都快崩溃了，更恐怖的是还要出差（要知道当初我从实施转开发就是对无休止的出差反感了），感觉再继续下去自己就废了，于是乎跟直属领导谈了几次心，好在领导人真的很好，立马就跟上面反馈了我的情况，然后经过另一个领导的一次谈话，我直属领导的几次努力，再加上曲折的过程，我顺利地从Android coder转为Java Web coder。在此感谢所有帮助过我的人，感谢你们长的这么好看还对我辣么好。

其实前期刚转过去的时候是很痛苦的，Android跟Java Web完全是两个体系，非要说他们的共同点那估计就是他们都是用的Java语言吧（貌似现在Google在Android方面大力推广Kotlin），所以初期要学的东西太多了，JavaScript，jQuery，Spring，SpringMVC，Mybatis…反正那会也是赶鸭子上架，直接上项目，遇到什么不会就查什么，现学现用，好在慢慢坚持下来了。很庆幸那会办公区域还没有断网（公司客户比较特殊，有保密要求，所以办公区域没网，有专门的上网区域查资料，但机子比较少，经常不够用），所以也学了不少东西。后来么，保密要求，断网了，要查东西去上网区，并且查到的资料如果想带进办公区域超级麻烦，要刻盘，经过中间机等等一系列，工作效率大打折扣，当然还有大打折扣的就是学习的欲望。本来之前看到一个技术点，复制粘贴往百度一扔，一堆资料就出来了，分分钟就搞定，现在的话得跑去上网机查，当然去了还不保证有机子，唉，所以后期基本上也就懒得去查资料，除非是项目上需要。由于断网，所以博客也就懒得写了，想想真够颓废的。

颓废的另一个主要原因就是玩游戏，由于之前追了12年的火影忍者，那也是唯一一个我一直追到结局的动画，所以有很多情怀在里面。年初刚好手机换成了7P，看到一同事在玩火影手游，被那精美的画面吸引了，刚好也是腾讯的游戏，也就跟着下了开始玩，当初想着冲着这个情怀也要体验下，就这么入坑了。哪想到这游戏炒鸡坑啊，氪金不说，还炒鸡费时间，没完没了的日常，每天要占据两个左右小时的时间，我又想把它玩好，所以业余时间的80%都浪费在这个上面了，上班休息的时候玩，下班回家也玩，下班时间的学习完全停止，博客停更的罪魁祸首也是这个，就这么浑浑噩噩一年过去了，好在过年期间我竟然把这个游戏戒掉了，一方面是因为觉得新的一年确实不应该颓废了，另一方面是这个游戏目前就我一人在玩，同事都在玩农药，游戏么，一个人玩就成单机游戏了，确实没啥意思了。2018年可能还会玩游戏，但不会像这样了，要玩也是那种不怎么氪金的，大家一起玩的游戏，不用天天登录，只需要想起来了上去玩一把那种的，更多的时候还是要留给实现自己的计划。

我的2018

废话不多说，来点干货：

  • 先把2017没完成的补上，主要是学会ps并熟练掌握，年底要能熟练运用，2017年就是这个愿望，结果只是将ps装到电脑上后就没了下文，2018，一定要完成。
  • 学习Java Web相关知识。重点掌握jQuery，Spring，SpringMVC，Mybatis，angularJS，SQL技能的提升，争取到年底达到熟练使用并了解一些实现原理。
  • 博客更新，虽然时间很紧张，加班很多，但，还是要坚持更新，也就不说今年非要更新多少篇了，随心随性就行。
  • github的更新与维护。以后估计大部分更新的都是Java Web相关的，当然Android的如果精力允许的话也会更新。

写在最后

努力了不一定能得到你想要的，但不努力一定得不到你想要的，诸君共勉。

希望等到2019年写这个的时候不要有太多的悔恨。
